# README

## Metodología de Desarrollo

Para este proyecto hemos implementado **Trunk-Based Development (TBD)**. 

Al tratarse de un proyecto de escala pequeña, esta metodología nos permite:
* **Integración continua:** Evitamos ramas de larga duración que generen conflictos complejos.
* **Agilidad:** Los cambios se integran rápidamente a la rama principal (`main`), facilitando el despliegue.
* **Simplicidad:** Reduce la sobrecarga de gestión de ramas, ideal para equipos pequeños o desarrollo individual.
